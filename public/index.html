Axum 0.7 + SQLx (SQLite) + tower-http

Rutas útiles:

  • GET / — saludo simple
  • GET /health — health check
  • GET /api/users — lista usuarios
  • POST /api/users — crea usuario { "name": "Alice" }
  • GET /api/users/:id — obtiene usuario por UUID
  • DELETE /api/users/:id — elimina usuario
